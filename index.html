/mcp/connect를 거쳐 발급한 유저는 42명 중 일부에 불과하고, 나머지는 메인 페이지에서 바로 키 발급한 것. 즉, 가이드/스킬/커넥트 페이지를 안 봐도 발급 자체는 가능한 구조.

그러나 발급 ≠ 사용. 101명 중 실제 크레딧을 소진한 유저는 45명(44.6%)뿐. 메인에서 "일단 발급"은 쉽지만, 발급 후 실제 MCP 연동까지 가는 것이 진짜 허들.

/mcp/connect를 방문한 42명은 참여 시간 1분 18초로, 진지하게 연동을 시도한 유저. 이들의 발급→사용 전환율이 메인에서 바로 발급한 유저보다 높을 것으로 추정.

활성 사용자 기준, 메인 7,454명 중 가이드까지 1,411명(19%), 스킬 497명(7%)까지는 탐색하지만, 연동 페이지(/mcp/connect)에 도달하는 유저는 42명(0.56%)에 불과.

그러나 메인에서 바로 키 발급이 가능한 구조이므로, connect 미경유 발급이 오히려 다수(101명 중 대부분). 진짜 문제는 connect 도달이 적은 게 아니라, 발급 후 실제 MCP 연동·사용까지 가는 유저가 45명(44.6%)뿐이라는 것.

이탈 원인 추정:

① 메인에서 "일단 발급" → 연동 방법을 모르거나 나중에 하려다 안 함
② 연동 페이지(connect)까지 간 42명은 참여 1분 18초로 진지하게 시도했으나, 터미널/설정의 복잡성에서 일부 좌절
③ "재밌어 보이지만 나한테 필요한 건 아니다" — SNS 유입의 일반인이 호기심으로 발급만 하고 끝

왜 "구조적 해소"인가

현재 메인 페이지에 이미 Kiro, Claude Desktop, Claude Connector, ChatGPT APP 4개 연동 옵션 버튼이 배치되어 있습니다. 문제는 이 버튼을 눌러도 현재는 수동 설정이 필요한 것.

GPT 앱/클로드 커넥터가 정식 등록되면:

• 메인 페이지 "ChatGPT APP" 버튼 → GPT 스토어 직행 → 1클릭 설치 → 바로 사용
• 메인 페이지 "Claude Connector" 버튼 → 커넥터 추가 → 바로 사용
• 가이드/스킬/커넥트를 거치지 않고도 메인에서 바로 전환 가능한 숏컷 퍼널이 생김

즉, 현재 4단계(메인→가이드→스킬→커넥트→발급→설정→사용)가 2단계(메인→GPT 앱 설치→사용)로 축소.

단, GPT 앱/커넥터가 도입되어도 BASIC 1년 데이터 제한 문제는 그대로 남음. 연동이 쉬워져 더 많은 BASIC 유저가 유입되면, 1년 제한으로 인한 이탈이 오히려 부각될 수 있음.

"말과 행동의 괴리" 분석

긍정 반응 5곳 중 실제 지속 사용은 1곳뿐

미팅에서 긍정 5곳 중 의미 있는 사용량 + 복수일 사용을 보인 곳은 그린브릭스(339, 4일) 1곳. "말"과 "행동"이 일치하지 않는 것은 B2B 영업의 전형적 패턴이나, 원인 분석이 필요.

유형기업실 행동원인 분석
말 = 행동그린브릭스4일 339cr AI 전담자가 미팅 참석 + 즉각적 Pain point(프로젝트 수량 문제) 존재. 전담자 + 명확한 Pain point = 실사용.
말 ≫ 행동TBWA2일 27cr "즉시 연동, 상당한 관심" → 실 사용 미미. 일부 팀만 AI 가능한 환경에서 전사 확산이 안 됨. 또는 연말까지 보류 중.
말 ≫ 행동허블2일 54cr "업무 자동화에 요긴" → 2일 후 멈춤. 구체적 use case 부재 또는 예의상 긍정이었을 가능성.
미팅 직후 집중와이즈버즈1일 252cr 미팅 당일 대량 사용 = 교육 중 즉석 실습. 이후 복귀 여부가 핵심 (아직 3일밖에 안 지남).
보안/승인 막힘이노션1일 88cr "보안 이슈로 컨펌 필요" → 실제로 1회 후 중단. 보안 승인이 막혔을 가능성 높음.
"급하지 않다"빗썸·팀민트미사용 "당장 급하지 않다" = 자발적 동기 부재. Push 없으면 무기한 보류.
실사용 예측 시그널

실사용으로 이어지는 조건:
① AI/데이터 전담자가 미팅에 참석 (그린브릭스 ✓)
② 당장의 구체적 Pain point 존재 (프로젝트 수량 문제, 여러 앱 비교 필요 등)
③ 클로드/MCP 사용 환경이 이미 갖춰져 있음

실사용으로 이어지지 않는 시그널:
① "연말에", "나중에", "여유 생기면" = 현재 동기 없음
② "보안 이슈", "컨펌 필요" = 조직 프로세스에 막힘
③ "관심은 있다" + 질문 없음 = 예의상 긍정 (버즈빌)
④ 전사가 아닌 일부 팀만 AI 사용 = 내부 확산 동력 부재 (TBWA)

서비스 건강도 평가

유료 고객 중심으로 보면: 견조하다

유료 고객(프리미엄+에센셜) 16명 중 8명(50%)이 복수일 사용하고 있으며, 이 중 3명은 완전 정착 상태. 매일 쓰는 서비스가 아닌 특성을 감안하면, D+7 리텐션 36%(유료 기준)는 양호한 수치.

특히 알토스 VC는 최근 가입(6/22) 후 5일 연속 사용으로 가장 건강한 온보딩 패턴을 보여주며, 이는 "여러 앱을 봐야 하는" 유저에게 제품 가치가 작동함을 증명.

BASIC(무료) 기준으로 보면: 문제 있다

D+1 리텐션 18%, D+3 이후 0%는 명확한 이탈 신호. 분석 기간 내 30크레딧/일로 운영(6/29부터 100으로 상향 예정). 30크레딧은 MCP 분석 1~2회로 소진되는 양이어서 가치 체감 자체가 어려웠음. 최근 1년 데이터 조회 불가 제약도 이탈 원인으로 추정.

100크레딧 상향 후 리텐션 변화를 주시해야 하며, 변화 없으면 1년 제한이 본질적 문제.

미팅 직후형(C그룹) 4명: 좀 더 지켜봐야 한다

그린브릭스·와이즈버즈·허블은 미팅에서 긍정 반응을 보였으나 이후 중단. 이들은 "시기가 안 맞아서" 일수도 있고 "연동이 귀찮아서" 일수도 있음. GPT 앱/커넥터 출시 시 자연 복귀 여부로 진짜 니즈를 판별 가능.

TBWA는 특이 케이스 — "즉시 연동, 상당한 관심"이었으나 2일 사용(총 27크레딧) 후 완전 중단. 연말 제안 시즌까지 기다리는 것인지, 아니면 기대 대비 임팩트가 적었는지 팔로업 필요.

결론

한 줄 요약

유료 고객 기반은 견조하다. 제품 가치는 증명되었다. 무료 유저 리텐션과 기술 허들이 확장의 병목.

상세

유료 고객 50% 복수일 사용, D+7 리텐션 36%는 "매일 쓰는 서비스가 아닌" 제품 특성을 감안하면 건강한 초기 지표입니다. hsad(13일), 한국투자증권(6일), 알토스(5일 연속), 휴맥스(8일) 등 정착 유저가 확보되었고, 이들의 사용 패턴은 실질적 업무 활용을 보여줍니다.

반면, BASIC D+3 리텐션 0%는 크레딧 상향만으로는 해결되지 않는 구조적 문제(1년 데이터 제한 또는 연동 허들)를 시사합니다. 그랜드 오픈 시 GPT 앱으로 연동 허들이 해소되면 BASIC 리텐션 변화를 주시해야 하며, 변화 없으면 1년 데이터 제한 완화를 검토해야 합니다.

GPT 앱/커넥터 출시가 핵심 변수이며, 출시 후 2주 내 ① C그룹(미팅 직후형) 복귀율, ② BASIC D+1 리텐션, ③ 신규 정착 유저 속도를 집중 모니터링할 것을 권장합니다.
