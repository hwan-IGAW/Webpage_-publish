모바일인덱스 3.0

지표 서비스에서 의사결정 플랫폼으로의 전환

IGAW | 대표 보고용 | 2026년 6월

Executive Summary

모바일인덱스는 현재 시장에서 '지표 서비스'로 인식되고 있습니다. 본 기획안은 이를 기업의 의사결정 도구 수준으로 격상시키기 위한 전략 방향을 제시합니다.

핵심 전환 축은 ① 시장 분석 툴, ② 시장 모니터링 도구, ③ 액션 툴(MI Ads·디파이너리 연계)의 3가지이며, MI 자체는 분석 + 모니터링에 집중합니다.

MCP(Model Context Protocol) 기반 데이터 융합을 통해 파편화된 이종 데이터를 통합하고, 2027년 1월 MI 3.0 출시를 목표로 합니다.

현황: 파편화된 데이터 생태계

모바일인덱스는 다양한 이종 데이터를 보유하고 있으나, 각각 별도 상품으로 서비스되어 종합적 검토가 어려운 상황입니다.

3

게임 인덱스

+
1st Party 데이터 연결

애드브릭스·디파이너리 연계 시 광고 성과 데이터 등

TV 광고 집행량 + 앱 사용량 변화 + 카드 결제액(앱마켓 매출) 변화
⬇
광고 → 사용자 유입 → 매출 전환까지 종합 분석 가능

그러나 현재는 데이터 파편화로 인해 현실적으로 이루어지지 않고 있음

해법: MCP 기반 데이터 융합

MCP(Model Context Protocol)가 해결하는 것

MCP화 되는 순간 데이터들의 융합이 매우 수월해지며, 확장된 관점 제시가 손쉽게 가능해집니다.

  • 모바일인덱스 MCP — 출시 완료 (앱 사용량)
  • TV 인덱스 MCP — 7월 중순 출시 예정
  • 소비 인덱스 MCP — 8월 개발
  • 앱 마켓 인덱스 + 게임 인덱스 MCP — 9월 개발

추가 데이터 확장: 외부 공공/민간 데이터 연동

당사 MCP와 결합 가능한 무료 MCP 또는 스킬셋을 함께 공급하여, 분석의 범위를 대폭 확장합니다.

📊
공공 데이터

주가, DART 공시, 통계청, 국토부, 기상청

💬
민간 데이터

SNS/커뮤니티, 쇼핑몰 리뷰 데이터

🧪 합성 소비자 인텔리전스(SCI) — Alpha 개발 완료

현재 IGAW에서 SCI 알파 버전 개발을 완료한 상태로, 이 데이터가 추가될 경우 시장 분석 역량의 비약적 확장이 기대됩니다.

모니터링의 2대 기능

Detection
변화의 포착

즉각 인지가 필요한 급변화부터, 데일리·먼슬리 지표만으로는 파악이 안 되는 완만한 구조적 변화까지 포착

Interpretation
변화의 해석 제시

분석 툴(다차원 데이터)을 적극 활용하여 변화의 원인과 영향을 자동 해석

4

MI 3.0 제품 구성(안)

분석 + 모니터링을 결합한 통합 의사결정 플랫폼

구성 요소 주기 핵심 기능 비고
데일리 리포트 Daily 변화량 중심 모니터링 스냅샷 클라이언트·경쟁사·업종
위클리 리포트 Weekly 데일리 확장 주간 트렌드 데일리 리포트의 확장 버전
월간 리포트 Monthly 스냅샷 분석 트렌드 분석 변화 원인 해석 전월 대비 + 전년 동기 대비
분기 리포트 Quarterly 트렌드 분석 원인 해석 업종 단위 심층 분석
연간 리포트 Yearly 딥다이브 분석 소비·앱마켓·TV 통합 유료 판매 (200~400만원)
커스텀 대시보드 On-Demand 대화형 셀프 구성 계정당 n개 페이지 대화형으로 자유 구성
MCP 비정형 분석툴 On-Demand 대화창 AI 분석 자유로운 질의 기반 분석
구버전 보기 Legacy 메뉴별 페이지 기존 사용자 호환
